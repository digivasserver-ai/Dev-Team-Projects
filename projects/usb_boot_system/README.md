# Universal USB Boot System

## Overview
A universal USB boot system that supports Android, Windows, Linux, and more.

## Features
- Multi-platform support (Android, Windows, Linux, etc.)
- Native tool integration
- Multi-language database
- Ventoy integration

## Roadmap
- Map out boot options for all platforms
- Store native tools and configurations
- Implement multi-language database
- Integrate Ventoy for live boot

## SSH Concept Test
### Objective
Test the feasibility of using GitHub Codespaces as a remote terminal environment for the USB boot system project.

### Steps
1. **Create a Codespace** for the `Dev-Team-Projects` repo.
2. **SSH into the Codespace** and verify remote access.
3. **Test SSH-based workflows** (e.g., editing files, running commands).
4. **Document findings** in `projects/usb_boot_system/ssh_test.md`.

### Expected Outcomes
- **Pros**: High-spec hardware, isolated environment.
- **Cons**: Latency for local tasks (e.g., ADB/USB debugging).

### Next Steps
- **Approve** the SSH concept test.
- **Proceed** with implementation if approved.
